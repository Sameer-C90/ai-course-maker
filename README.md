#AI Course Maker

A MERN-stack project which uses gemini API on backend to generate AI courses based on prompt inputted by user.

##Description

This project generates a 5 module structure based course that is shown as a stylized accordion. The modules are in markdown and they are stylized with the help of react-markdown component. There is a Learn More Button to increase course modules in increments of 5 modules. There are Login, Signup and Dashboard pages for Authentication and User-based course storage. The Dashboard displays the courses generated by a User and they can be deleted or opened again to see course info.

##Tech Stack

- MERN Stack (Mongodb, ExpressJS, ReactJS by Vite, NodeJS)
- React Router for routing
- JWT and AUTH Setup with jsonwebtoken and Mongoose for Mongodb
- Axios for requests
- TailwindCSS for styling
- Gemini API and google/genai for AI based Course Generation

##Installation

```bash
    git clone 
    cd 
    cd server
    npm install
    cd client
    npm install
```

##Setup

```bash
    cd server
    touch .env
```
Add these env variables for local use of the backend API.
```
    PORT=<custom port for hosting>
    GEMINI_API_KEY=<gemini api key for course generation>
    MONGO_URI=<mongo uri for custom mongodb storage>
    JWT_SECRET=<a secret keyword to make jwt secure>
    JWT_LIFETIME=<jwt lifetime>
```
To setup backend url for frontend follow this -
```bash
    cd client
    touch .env
```
Now add the backend url where it's hosted
```
    VITE_API_URL=<backend host>
```
